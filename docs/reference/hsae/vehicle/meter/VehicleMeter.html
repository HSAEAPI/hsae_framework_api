Class Overview

The VehicleMeter class implement VehicleManager sub service hsae.vehicle.VMSConstant.HSAE_VEHICLE_METER.
The VehicleMeter instance is create by VehicleManager, we use VehicleManager to make sure VehicleMeter only construct once for each client apk
all development can get this sub service by this way:

Example: 

     VehicleManager vehicleManager = (VehicleManager) getSystemService(HsaeContext.VEHICLE);
     VehicleMeter vehicleMeter = (VehicleMeter) vehicleManager.getVehicleService(VMSConstant.HSAE_VEHICLE_METER);

 This class provide two kind method, get/set value method and register/unregister VehicleMeterListener method.
