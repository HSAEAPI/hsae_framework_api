public static final int WIRELESS_CHARGE

Added in API level 2

Description: this Constant id is use to get and listen WIRELESS_CHARGE state.
Type : INT32;
Access : READ
Value :
0 : standby mode
1 : Charging
2 : Over-voltage protection
3 : Low-voltage protection
4 : Foreign body protection
5 : Over Power Protection
6 : Over-temperature protection
7 : Interrupted by PEPS
8 : Other fault
9 : charge complete (Support by the phone has full of reminders so the wireless charge will respond)
10 : Communication with wireless charging is lost
11~14 : reserve
15 : invalid
-1 : invalid
