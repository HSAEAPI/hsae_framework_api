hsae.net.wifi.HsaeWifiManager.HsaeSoftApCallback

Class Overview

Base class for soft AP callback. Should be extended by applications and set when calling {HsaeWifiManager.registerHsaeSoftApCallback(HsaeSoftApCallback, Handler)}.
