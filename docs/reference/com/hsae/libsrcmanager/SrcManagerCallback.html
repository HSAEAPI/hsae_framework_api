public void playPrevOrNext (int app, int pre, boolean isHardKey, int repeatCount)

Added in API level 2

Description :
Notify client (HMI) to play previous or next item from status bar or hard key.

Note:
If “isHardKey” is true, the field "reason" need be filled in 1 in JSON string "hu_play_start"
if “isHardKey” is false, it need be filled in 0.

Parameters
app int: the app mean application Id, the value of application Id refer to HsaeSrcMgrConstants
pre int: Play previous or next item. pre valid value list: 0: previous, 1: next.
isHardKey boolean: isHardKey is true: pressed prev/next hard key, otherwise false. true: pressed previous/next hard key
repeatCount int: the short/long press flag, short press flag : -1 long press flag : 0(long press start)--repeatCount(long press end)
Throws
RemoteException
