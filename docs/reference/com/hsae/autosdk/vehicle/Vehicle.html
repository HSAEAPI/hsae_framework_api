public int[] getTpmsStatus ()

Added in API level 1

Description :
Get the TPMS status

Returns
int[] the TPMS status,refers to Value.

Value:
[0][1] --front left tire pressure: 0 is normal, others are abnormal
[1][*] --front right tire pressure
[2][*] --rear left tire pressure
[3][*] --rear right tire pressure
Note:
One dimensional array corresponds to binary array value (array subscript 0 ~ 7) [00] [01] [10] [11] [20] [21] [30] [31]
Do not loop to get the TPMS status to avoid excessive CPU usage.

RemoteException
