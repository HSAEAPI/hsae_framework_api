package

com.hsae.autosdk.vehicle

This package contains Vehicle class that provides vehicle property and status API,the developers can use these API to get vehicle of property and status.

Example:
Get the state if key is out of the car 
    Vehicle mVehicle = (Vehicle)mContext.getSystemService(HsaeContext.HSAE_VEHICLE);
    boolean carKeyLeave_state = mVehicle.getKeyLeaveCarState();

Example:
Get the doors state 

    Vehicle mVehicle = (Vehicle)mContext.getSystemService(HsaeContext.HSAE_VEHICLE);
    Boolean[] door_state = mVehicle.getDoorState();
