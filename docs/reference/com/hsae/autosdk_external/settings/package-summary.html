package

com.hsae.autosdk_external.settings

This package provides vehicle auto setting class and intenface,the developers can use these API to develop related programs.

The AutoSettings_external calss is about client control server interface.

Example:
get the DA power state 
    AutoSettings_external mAutoSettingsExternal= (AutoSettings_external) mContext.getSystemService(HsaeContext.HSAE_AUTO_SETTINGS_EX);
    boolean power_state = mAutoSettingsExternal.getPowerState();

The AutoExternalListener and AutoSettings_external.Auto_externalListener is an interface that can provide callback method. The development should implements this interface and register it .
