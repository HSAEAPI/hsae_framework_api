Package Index - Hsae_android_API_reference

1、Overview

This document describes the elements of the HSAE API, such as VehicleManager classes, that a developer is likely to use or extend. While the goal of HSAE API is to provide a complete solution for test Android applications and provide API for android developer.The HSAE API needs to be used with the HSAE SDK,the developers get the SDK through the following ways.

Packages

com.hsae.autosdk.settings

This package contains AutoSettings class that provides AutoSettings property and status API,the developers can use these API to get vehicle of property and status.
com.hsae.autosdk.vehicle

This package contains Vehicle class that provides vehicle property and status API,the developers can use these API to get vehicle of property and status.
com.hsae.autosdk_external.settings

This package provides vehicle auto setting class and intenface,the developers can use these API to develop related programs.
com.hsae.libsrcmanager

This package contains interfaces of source manager, the interfaces are based on Android AIDL.
hsae.content

This package contains only HsaeContext class that provides many string constants and methods,the developer use these HsaeContext API to develop related programs.
hsae.net.wifi

This package provides operation of softAp,the developers can used to get softAp state and number of connected clients.
hsae.vehicle

This package contains many important subpackages and many core classes that support the whole SDK function, the developers use this package and subpackages to develop function.
hsae.vehicle.ac

This package provides operation and control of vehicle air conditioning,the developers can used to manager vehicle air conditioning.
hsae.vehicle.ivi

This package provides the class of data operation related to HsaeDeviceInfo and HsaeDeviceInfo API for android developer, the developer use these vehicle API to develop related programs.
hsae.vehicle.light

This package provides the class of data operation related to VehicleLight API for android developer, the developer use these vehicle API to develop related programs.
hsae.vehicle.meter

This package provides the class of data operation related to vehicleMeter and vehicleMeter API for android developer, the developer use these vehicle API to develop related programs.
hsae.vehicle.power

This package provides the class of data operation related to VehiclePowerManager API for android developer, the developer use these vehicle API to develop related programs.
